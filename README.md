# Bonfire Math Seminar Materials

This serves as a repository of programming materials for the Calculus I seminar as part of the **2026 Engineering Summer Bridge Program**. The purpose is to provide foundational introduction material for `SymPy`, something we felt was lacking in the existing curriculum.

## How to navigate

Workshop materials are provided as Jupyter notebooks (`.ipynb` files) in the `notebooks` directory.

The `reference_materials` directory contains quick-reference sheets covering essential SymPy methods, syntax, and examples.

The `resources` directory contains additional Calculus I and engineering materials for further study.

We encourage you to continue exploring SymPy independently. For more information, visit the [official SymPy documentation](https://docs.sympy.org/latest/index.html).
