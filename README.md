# Bonfire Computational Calculus Materials

This repository contains computational activities for the Calculus I seminar in the **2026 Engineering Summer Bridge Program**. The workshops introduce Python tools that help students explore calculus through symbolic computation, visualization, and applied problem-solving.

## How to navigate

Workshop materials are provided as Jupyter notebooks (`.ipynb` files) in the `notebooks` directory.

The `reference_materials` directory contains quick-reference sheets covering the Python libraries used in the workshops, including essential methods, syntax, and examples.

The `resources` directory contains additional Calculus I and engineering materials for further study.

We encourage you to continue exploring these tools independently. For more information, visit the official [SymPy](https://docs.sympy.org/latest/index.html) and [Matplotlib](https://matplotlib.org/stable/) documentation.
